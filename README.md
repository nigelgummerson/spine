# Spine Planner

**Spinal Instrumentation Plan & Record** — a web application for pre-operative spinal surgery planning.

## Live Application

**[plan.skeletalsurgery.com/spine](https://plan.skeletalsurgery.com/spine/)**

## Features

- Visual spinal construct planning (Oc–Pelvis)
- Pedicle screws, hooks, cages, osteotomies, corrective forces
- Population-based screw size ranges with sex-specific morphometry
- Screw trajectory system with published per-level angles
- Automated implant inventory generation
- PDF and JPEG export for clinical records
- Dual-window sync for operating theatre displays
- 22 languages (16 European + Hindi + Arabic + Hebrew + Chinese + Japanese + Korean)
- Fully offline — works from a USB stick or local file

## Standalone Download

For offline use without a web connection, download the standalone HTML file from [Releases](https://github.com/nigelgummerson/spine/releases).
